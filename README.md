# BuscadorDemo
Buscador Empresarial Inteligente (LabSearch Pro). Prototipo de motor de búsqueda federado y contextual full-stack. Incluye arquitectura backend para orquestación de múltiples fuentes (APIs + S3), lógica de expansión de relaciones, resaltado de sintaxis y frontend reactivo con soporte de temas (Dark/Light).
